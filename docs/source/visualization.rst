Visualization
-------------
The `visualization` directory contains backend python files to create data visualizations.

* `helper_functions.py`: contains functions to aid in the generation of visualizations
* `visualization_execution.py`: contains the method `purchase_sale_vs_time()` which is responsible for generating a visualization
